Get acces to animal facts API

If you are looking for a free, easy way to get facts about different animals you can sign up for our API whitelist. If you are interest, please send us an email to api@cseteandor.hu, with a subject of "API whitelist request", and we'll contact you as soon as possible. You'll receive your API key in email, what you can use to request facts from our database. Please read the full documentation for more informations!
