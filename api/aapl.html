Apple részvények (AAPL) aktuális árfolyamának megjelenítése dollárban, majd átváltása forint árfolyara, az aktuális átváltási ráta használatával. (percenként maximum 5 kérelmezéssel)

Apple részvény

AAPL

A program működése az https://www.alphavantage.co weboldal ingyenes részvény API szolgáltatásra alapul. Itt maximum 5 kérelmezés lesz végrehajtva percenként, a további kérelmezés végrehajtásokat nem támogatja az ingyenes fiók. Az alphavantage felel az AAPL részvények aktuális USD árfolyamának lekérdezéséért, és az aktuális USD-HUF váltási ráta lekérdezéséért is, (ezzel kell megszorozni az AAPL-USD árfolyamot, hogy megkapjuk az AAPL-HUF árfolyamot.). Majd a lekérdezett adatokat kiírja a weboldalon. A HIBA feliratot a nemmegfelelő internetkapcsolat, vagy a sok kérelmezés okozza. Ebben az esetben próbáld újra egy perc múlva.
